## 🪑 CADEIRA E POSTURA

### Cadeira Ideal

**Especificações:**
- Altura ajustável: 45-55cm (da base ao assento)
- Encosto: Suporta região lombar
- Rodízios: 5 pontos (estável)
- Material: Respirável (não superaquece)
- Tipo: Giratória 360°

**Modelos recomendados:**
```
Faixa 1 (Básico): R$300-600
├─ Cadeira de escritório padrão
├─ Sem ajuste de altura fina
└─ OK para uso ocasional

Faixa 2 (Profissional): R$600-1500
├─ Cadeira ergonômica com suporte lombar
├─ Múltiplos ajustes
└─ RECOMENDADO para uso diário

Faixa 3 (Premium): R$1500+
├─ Cadeira de laboratório high-end
├─ Até 24h conforto
└─ Para pesquisa intensiva
```

**Ajustes corretos:**
```
      Olhos
```

## 🖥️ POSICIONAMENTO DO PC E MONITORES

### Setup com 1 Monitor (Básico)

```
        MICROSCÓPIO
            ↓
    ┌──────────────┐
    │              │
    └──────┬───────┘
           │
      MONITOR USB
      (Câmera do microscópio)
           │
    ┌──────┴───────┐
    │              │
    │   ROBÔS      │
    │  Síntese +   │
    │  Injeção     │
    │              │
    └──────────────┘
           ↑
       PC + Controle
```

**Posicionamento:**
- Altura do monitor: Olho no topo da tela
- Distância: 50-70cm dos olhos
- Ângulo: Ligeiramente abaixo (15-20°)
- Lateralidade: Direto (não virado)

---

### Setup com 2-3 Monitores (Profissional)

```
MELHOR LAYOUT:

    ┌─────────────┬─────────────┬─────────────┐
    │ MONITOR 1   │ MONITOR 2   │ MONITOR 3   │
    │ (Primário)  │ (Central)   │ (Auxiliar)  │
    │ GUI         │ Microscópio │ Dados/Logs  │
    │ Controle    │ Visão real  │ Estatísticas│
    └─────────────┴─────────────┴─────────────┘
           ↑            ↑            ↑
        Curvo          25cm         Lateral
        (você)

Altura: Todos alinhados (olho no topo)
Distância: 50-70cm do olho
Ângulo: Monitor central reto, laterais 25-35°
```

**Especificações dos monitores:**

```
Monitor 1 (Primário - GUI):
├─ Tamanho: 27" (ideal para 1920x1080)
├─ Taxa: 60Hz mínimo
├─ Painel: IPS (cores precisas)
├─ Resposta: 5ms (OK para dados)
└─ Uso: Controle do sistema

Monitor 2 (Microscópio - Câmera):
├─ Tamanho: 24" (para visão nítida)
├─ Taxa: 30fps+ (câmera)
├─ Painel: IPS (precisão de cores)
├─ Resposta: 1ms (melhor)
├─ Entrada: USB ou HDMI
└─ Uso: Visão em tempo real da injeção

Monitor 3 (Auxiliar - Logs):
├─ Tamanho: 21-22" (economia)
├─ Taxa: 60Hz
├─ Painel: TN (suficiente)
├─ Uso: Histórico, alertas, dados
└─ Opcional (pode usar tablet)
```

## 🤖 POSICIONAMENTO DOS ROBÔS

### ROBÔ 1: Síntese de DNA

```
            ALTURA: 100cm
                ↑
    ┌───────────────────────┐
    │                       │
    │  SÍNTESE DE DNA       │  ← Acima de você
    │  ┌──────────────────┐ │
    │  │ Câmara reação    │ │
    │  │ Bombas/Válvulas  │ │
    │  │ Eletrônica       │ │
    │  └──────────────────┘ │
    │                       │
    └───────────────────────┘
           ↑
        SUPORTE
        FIXO
           ↑
        BANCADA
        ← 30cm do seu olho
```

**Posicionamento ideal:**

```
Altura:
├─ Parte inferior: Nível do seu olho (60-70cm acima da mesa)
├─ Câmara: Ligeiramente acima (até 120cm)
└─ Acesso visual: Completo

Distância:
├─ Frontal: 30-50cm (fácil acesso)
├─ Lateral: 20cm (não atrapalha injeção)
└─ Altura: Permita visualização completa

Ventilação:
├─ Espaço traseiro: 10cm (ar circula)
├─ Lateral: 5cm mínimo
├─ Não cubra ventilação
└─ Mantenha sensor de temperatura visível
```

### ROBÔ 2: Injeção de Genoma

```
            ALTURA: 60cm
                ↑
    ┌───────────────────────┐
    │                       │
    │  MICROSCÓPIO +        │
    │  ROBÔ INJEÇÃO         │
    │  ┌──────────────────┐ │
    │  │ Microscópio      │ │
    │  │ Motores XYZ      │ │
    │  │ Porta-ovo        │ │
    │  │ Agulha           │ │
    │  └──────────────────┘ │
    │                       │
    └───────────────────────┘
           ↑
        SUPORTE
        ARTICULADO
           ↑
        BANCADA
        ← Seu olho aqui
```

**Posicionamento ideal:**

```
Altura microscópio:
├─ Ocular: Ligeiramente abaixo do seu olho (10-15°)
├─ Campo: Direto para frente
└─ Sem torcer pescoço

Distância:
├─ Microscópio: 20-30cm do seu olho
├─ Eixo óptico: Perpendicular ao seu corpo
└─ Espaço lateral: 15cm mínimo

Motores:
├─ Posição: Atrás/lateral do microscópio
├─ Fora do caminho: Não atrapalha movimento do braço
└─ Controle: Via joystick ou botões (acesso fácil)

Agulha/Injetor:
├─ Posição: Acima do microscópio (visão clara)
├─ Ângulo: 45° em relação ao ovo
└─ Proximidade: 2-3mm do ponto de injeção
```

## 🌡️ AMBIENTE E CLIMA

### Temperatura

```
Ideal: 20-24°C (constante)

Por zona:
├─ Robô Síntese: 37.5°C (controlado internamente)
├─ Robô Injeção: 22°C (ambiente)
├─ PC: 18-25°C (não superaquece)
└─ Seu corpo: 21°C (conforto)

Equipamento necessário:
├─ Ar condicionado: 2 toneladas mínimo
├─ Isolamento térmico: Poliestireno nas janelas
└─ Termômetro contínuo: Monitorar
```

### Umidade

```
Ideal: 40-60%

Por zona:
├─ Robô Síntese: 75-80% (controlado)
├─ Ar ambiente: 45-55%
└─ Eletrônica: 30-70% OK

Equipamento:
├─ Umidificador: Se < 30%
├─ Desumidificador: Se > 70%
└─ Higrômetro: Monitoramento
```

### Iluminação

```
Ideal: 500-750 lux (trabalho de precisão)

Fonte principal:
├─ LED overhead: 4000-5000K (branco neutro)
├─ Intensidade: 400-600 lux
└─ Sem sombras no microscópio

Iluminação auxiliar:
├─ LED microscópio coaxial: Integrado
├─ Task light: 50-100cm para lado
└─ Anti-glare no monitor: Importante

Checklist:
├─ Sem reflexo na tela
├─ Campo de injeção bem iluminado
├─ Sem sombra da sua cabeça
└─ Conforto visual 8 horas+
```

## ⚠️ PONTOS CRÍTICOS DE SEGURANÇA

### Elétrica

```
├─ Piso: Condutor (terra correta)
├─ Circuito: Disjuntor 20A dedicado
├─ Tomadas: No mínimo 4 (uma por equipamento)
├─ UPS: Backup 2000VA mínimo
├─ Extensão: Não use extensão (plugue direto)
└─ Inspeção: Anual por técnico
```

### Segurança Física

```
├─ Robôs: Antivibração (não danifica equipamento)
├─ Cabos: Bem fixos (não tropeça)
├─ Prateleiras: Fixadas na parede (não cai)
├─ Agulhas: Recipiente biohazard acessível
├─ Saída emergência: Livre de obstáculos
└─ Extintor: Próximo e acessível
```

### Ergonomia

```
├─ Pausas: 15min a cada 1 hora
├─ Movimento: Não fique imóvel 2+ horas
├─ Postura: Ajuste diariamente
├─ Monitor: Altura correcta sempre
└─ Antifadiga: Tapete embaixo da cadeira
```
