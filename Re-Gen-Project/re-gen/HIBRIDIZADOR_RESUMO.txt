┌─────────────────────────────────────────────────────────────────────────────┐
│ 🎯 PROTOCOLO DE HIBRIDIZAÇÃO (Resumido)                                     │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│  1. INICIALIZAR                                                             │
│     └─ Sistema aquece para temperatura alvo (~2-3 min)                     │
│        LED: VERDE (pronto)                                                 │
│                                                                              │
│  2. CARREGAR AMOSTRAS                                                       │
│     └─ DNA-A: 500 µL (entrada 1)                                           │
│        DNA-B: 500 µL (entrada 2)                                           │
│        Buffer: 1000 µL (entrada 3)                                         │
│                                                                              │
│  3. CONFIGURAR PARÂMETROS                                                   │
│     └─ Temperatura: 37.5°C (ótima)                                         │
│        Duração: 4 horas (padrão)                                           │
│        Velocidade bomba: 75%                                               │
│                                                                              │
│  4. INICIAR REAÇÃO                                                          │
│     └─ Pressione [INICIAR] ou --terminal                                  │
│        LED: AZUL (em andamento)                                            │
│                                                                              │
│  5. MONITORAR (4 horas)                                                    │
│     └─ Acompanhe progresso em tempo real                                   │
│        • Monitor esquerdo: Parâmetros (T°C, pH, progresso)                 │
│        • Monitor direito: Gráficos de DNA                                  │
│        • Câmara: Visualização do estado                                    │
│                                                                              │
│  6. RESULTADO FINAL                                                         │
│     └─ DNA Hibridizado: ~180-200 µM                                       │
│        Eficiência: ~90-95%                                                 │
│        Arquivo: hibridizacao_YYYYMMDD_HHMMSS.json                         │
│                                                                              │
│  7. LIMPEZA                                                                 │
│     └─ Esterilização 95°C (5 min)                                          │
│        Resfriamento para 20°C                                              │
│        Pronto para próximo ciclo                                           │
│                                                                              │
└─────────────────────────────────────────────────────────────────────────────┘
