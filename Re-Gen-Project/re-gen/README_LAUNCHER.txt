🎮 COMO USAR O LAUNCHER
═══════════════════════════════════════════════════════════════════════════════

1. Menu interativo:
   ./launcher.sh

   Opções:
   [1] GUI - Interface gráfica
   [2] Síntese - Gerar genoma
   [3] Testes - Validar sistema
   [4] Documentação - Ler guias
   [5] Robôs - Info dos robôs
   [6] Dados - Ver arquivos
   [7] Ferramentas - Utilitários
   [0] Sair

2. Atalhos diretos:
   ./run_gui.sh              # GUI
   ./START.sh synth          # Gerar genoma
   ./START.sh synth "Trice"  # Genoma específico
   ./START.sh test           # Testes
